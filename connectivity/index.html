    Connectivity

    Estraad uses modern technologies that combine voice/video, player data networking and 3D graphics.

    In order for clients to connect to the platform, a network that supports the following is required:

    • Allow for WebSocket connections
    • Allow for ports 80 and 443 via TLS.
    • Allow for ports 49152-60999 on UDP and TCP. You will also need to open port 19302 on UDP and TCP.

    If your company does not support this, please try another network or use a VPN to connect without network restrictions.
